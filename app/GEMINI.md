
## App Module Agent Guide

This guide provides instructions for interacting with the Gemini agent within the `app` module.

### Purpose & Scope
The `app` module hosts Streamlit applications for visualizing model outputs and interacting with the prediction service.

### Gemini's Role
- **App Development:** Assist in creating and modifying Streamlit applications.
- **UI/UX:** Help with the design and layout of the applications.
- **Integration:** Connect the applications with the backend services in the `property_adviser` package.

### Interaction Guidelines
- **Specify the App:** When requesting changes, specify which application you are referring to (e.g., `predict_app.py`, `market_insights_app.py`, `market_price_visualise.py`).
- **Describe the UI:** Provide clear descriptions of the desired UI changes.
- **Data Sources:** If the changes involve new data, specify the source of the data and how it should be integrated.
