
## Core Module Agent Guide

This guide provides instructions for interacting with the Gemini agent within the `property_adviser/core` module.

### Purpose & Scope
The `core` module provides reusable utilities for logging, configuration, I/O, and other shared functionalities.

### Gemini's Role
- **Maintain Core Utilities:** Help in maintaining and extending the core utilities.
- **Promote Reusability:** Identify and promote reusable components from other modules into the `core` module.
- **Ensure Consistency:** Ensure that all modules use the core utilities consistently.

### Interaction Guidelines
- **Focus on Reusability:** When proposing new additions to the `core` module, explain how they can be reused across multiple modules.
- **Avoid Module-Specific Logic:** Do not add logic that is specific to a single module to the `core` module.
- **Backward Compatibility:** Ensure that any changes to the core utilities are backward compatible.
