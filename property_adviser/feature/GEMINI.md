
## Feature Module Agent Guide

This guide provides instructions for interacting with the Gemini agent within the `property_adviser/feature` module.

### Purpose & Scope
The `feature` module is responsible for scoring, curating, and selecting features for the model.

### Gemini's Role
- **Feature Engineering:** Assist in developing new feature engineering techniques.
- **Feature Selection:** Help in implementing and tuning feature selection algorithms.
- **Configuration:** Manage the feature selection configuration in `config/features.yml`.

### Interaction Guidelines
- **Explain the Logic:** When proposing new feature engineering or selection logic, explain the reasoning behind it.
- **Use Configuration:** Prefer to control the feature selection process through the `config/features.yml` file.
- **Mind the Outputs:** Be aware of the output files generated by this module and ensure they are in the correct format.
