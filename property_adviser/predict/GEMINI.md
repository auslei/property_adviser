
## Predict Module Agent Guide

This guide provides instructions for interacting with the Gemini agent within the `property_adviser/predict` module.

### Purpose & Scope
The `predict` module is responsible for making predictions using the trained model.

### Gemini's Role
- **Prediction API:** Help in developing and maintaining the prediction API.
- **Inference Logic:** Implement the logic for loading the model and making predictions.
- **Performance:** Optimize the prediction performance.

### Interaction Guidelines
- **Input and Output:** Clearly define the input and output schemas for the prediction API.
- **Model Loading:** Specify the path to the trained model to be used for prediction.
- **Error Handling:** Implement proper error handling for cases where the model or other artifacts are not found.
